# Iris — Start Here

You're designing the UX and visual system for Viridian P1 (Improv Mastery Tracker).

## Your Task

Create wireframes and design system for:

1. **Teacher workflows:** Class overview → Week detail → Student dashboard
2. **Student workflows:** Anchor standards dashboard → Skill detail → Self-rating form
3. **Design system:** Components, colors, typography, spacing
4. **Mobile responsiveness:** Desktop-first, fully responsive on phone

## Key Decisions (Locked)

✓ **Two hierarchies (P2 vision):** Skills (complex, assessed over time) + Content (unit-based, discrete)  
✓ **Anchor standards visual priority:** Emphasized (starred/bold), non-anchors secondary (collapsible)  
✓ **Self-eval workflow:** Student rates → submits → sees teacher rating → can revise (timestamps tracked)  
✓ **Discrepancy UI:** Prominently flagged, dropdown for teacher/student communication  
✓ **Assessment log integration:** Visual connection between assessment method + date + context  
✓ **K-12 parallel:** Mirror Improv structure, hierarchical standards tree (not flat list)
